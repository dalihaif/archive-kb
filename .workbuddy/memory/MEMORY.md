# 项目记忆

## archive-kb项目

### 项目概述
档案政策监控与知识库平台，使用Flask + SQLite + Scrapling + Bootstrap5开发。

### 数据库模型
- **MonitorSource**: 监控数据源（名称、URL、类型、分类、爬取方法等）
- **Policy**: 政策法规条目（标题、URL、摘要、内容、发布日期、发布机构等）
- **Category**: 知识库分类（支持父子分类）
- **KnowledgeItem**: 知识库条目（标题、内容、来源、标签等）
- **News**: 行业资讯条目（标题、URL、摘要、内容、发布时间、国家/地区、语言等）【新增】
- **User**: 管理员用户（用户名、密码哈希、关注关键词等）
- **MonitorLog**: 监控日志（数据源ID、状态、发现条目数、新增条目数等）

### 关键功能
1. **政策监控**：自动抓取政策法规，支持智能提取、详情页抓取、全文搜索
2. **知识库管理**：分类管理、条目管理、批量操作、智能分类建议
3. **行业资讯**：【新增】抓取国内外档案行业最新资讯，支持按国家/地区筛选
4. **定时任务**：使用APScheduler定时抓取政策法规和行业资讯
5. **关键词订阅**：用户可以订阅关键词，系统会检查新政策中是否命中
6. **数据备份/恢复**：支持数据库备份和恢复
7. **Excel导出**：支持将政策条目导出为Excel文件

### 爬虫策略
- **三级抓取策略**：
  1. 获取页面HTML
  2. 智能提取内容列表（支持配置CSS选择器或自动启发式提取）
  3. 抓取详情页全文（最多抓前5条）
- **降级方案**：Scrapling失败后立即降级到requests
- **去重**：使用SHA256哈希去重

### 全文搜索
- 使用SQLite FTS5实现全文搜索
- 支持Policy、KnowledgeItem、News三个表的搜索
- 优先使用FTS5搜索，失败后背降级到LIKE模糊匹配

### 已添加的数据源
- **政策数据源**：国家档案局、国家卫健委、国家法律法规数据库、中国政府网、中国医院协会等
- **资讯数据源**：【新增】国家档案局、中国档案学会、档案界、ICA、美国国家档案馆、英国国家档案馆、澳大利亚国家档案馆、加拿大图书档案馆、英国档案协会、美国档案工作者协会等

### 常见操作
- **启动应用**：`python app.py`（默认端口5050）
- **手动抓取政策**：访问`/admin/api/sources/<id>/crawl`
- **手动抓取资讯**：访问`/admin/api/news/crawl`或点击后台的"抓取资讯"按钮
- **查看日志**：应用启动时会在控制台输出抓取日志

### 注意事项
- 应用启动时会自动创建数据库表（如果不存在）
- 应用启动时会初始化预置数据源和知识库分类
- 后台无需登录即可访问（已移除登录限制）
- 定时任务默认间隔60分钟，可在config.py中修改CRAWL_INTERVAL_MINUTES配置
